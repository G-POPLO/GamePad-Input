GamePad Configuration

This is the configuration page for the GamePad extension.

Usage Instructions

  • Y Button (通常按钮索引为 3): 刷新当前页面。
  • B Button (通常按钮索引为 1): 关闭当前标签页。
  • Home Button (通常按钮索引为 0): 创建新标签页。
  • Left Bumper (LB) (通常按钮索引为 4): 切换到前一个标签页。
  • Right Bumper (RB) (通常按钮索引为 5): 切换到下一个标签页。
  • Left Trigger (LT) (通常按钮索引为 6): 后退历史记录。
  • Right Trigger (RT) (通常按钮索引为 7): 前进历史记录。
  • X Button (通常按钮索引为 2): 复制当前标签页。
